A clean web application using ASP.NET Core MVC, Entity Framework Core, Identity, SQL Server and Visual Studio 2022.

Use Entity Framework to perform CRUD (Create, Read, Update and Delete) operations and to implement advanced functionalities, such as pagination, search and sort.

Use Identity to implement user Registration, Authentication and Role-based Authorization. Besides, add many useful functionalities, such as edit the user profile, update the user password, display the registred users, update the user role, delete the user account, and request a password reset token.

Create the shopping cart and orders. Store the shopping cart into a cookie and  use colored badges to track the order status.
